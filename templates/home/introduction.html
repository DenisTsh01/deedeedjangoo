Welcome to DeeDee

DeeDee's purpose is to serve the user as a personal and virtual assistant.
DeeDee's features are face recognition, for easier and modern login, chat bot,
to make interactions with the user, and maybe in the future it can handle
complex vocal commands such as creating an event in the calendar for the user.
Also with DeeDee, the user can create tasks and events as reminders, types of
the events or deadline information are shown. DeeDee's wish is to help the user
as much as she can: so she provides basic weather information and also within
the wallet information data from the user, she can help him to see the total income
value or the payments or to display a better view of the flow of user's money.
